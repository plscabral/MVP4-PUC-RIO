### Pré-requisitos

* node

* npm
  ```sh
  npm install npm@latest -g
  ```

* yarn
  ```sh
  npm install --global yarn
  ```

### Instalação

1. Instalar as dependências do projeto
   ```sh
   npm install ou yarn install
   ```

### Execução

1. Rodar o comando abaixo:
   ```sh
   npm run dev ou yarn dev
   ```
